Role: Backend Developer - Core Business Logic

Entity: Store

Manages the physical store locations for the film rental system.

  • GET /api/stores - Retrieve a list of all stores
  • POST /api/stores - Create a new store location
  • GET /api/stores/{id} - Retrieve specific store details by ID
  • PUT /api/stores/{id} - Update store information

Entity: Staff

Manages the employees working at various store locations.

  • GET /api/staff - Retrieve all staff members
  • POST /api/staff - Register a new staff member
  • GET /api/staff/{id} - Retrieve staff details by ID
  • PUT /api/staff/{id} - Update staff credentials/info
